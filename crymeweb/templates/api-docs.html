What is CrymeClarity?

CrymeClarity is a RESTful API that serves public safety information for the city of Los Angeles.



API Documentation

POST /api/safety-analysis

The safety analysis endpoint is designed to return an analysis of the safety of a given location at a given time so long as the location is within the city limits of Los Angeles.

Request Properties
Property Type Description
Latitude Decimal The latitude of the location for which an analysis is desired
Longitude Decimal The longitude of the location for which an analysis is desired
Timestamp String The datetime for which the analysis is desired. (In format YYYY-MM-DDThh:mm:ss)

Response Properties
Property Type Description
Latitude Decimal The latitude of the location of the analysis
Longitude Decimal The longitude of the location of the analysis
Timestamp String The datetime of the analysis. (In format YYYY-MM-DDThh:mm:ss)
estimate.positive_prob Decimal The probability of a crime occuring within a half mile of the lat/lon provided within an hour of the timestamp provided.
estimate.negative_prob Decimal The probability of a crime not occuring within a half mile of the lat/lon provided within an hour of the timestamp provided.
Safety Rating String A qualitative safety rating of the location provided, at the time provided.
